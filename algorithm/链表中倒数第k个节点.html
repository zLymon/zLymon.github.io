# 链表中倒数第k个节点

输入一个链表，输出该链表中倒数第k个结点。

这题个人认为是需要死记解题方法的，因为看了挺久也不太理解其中的原理😂。。

解题思路：

  1. 定义快、慢指针指向头节点；
  2. 快指针向前走k-1步，慢指针不动；
  3. 快指针走完后，快慢指针一起走，当快指针走到尾节点时，慢指针指向的正是倒数第k个节点。

代码实现

function FindKthToTail (head, k) {
  // 判断特殊情况
  if (head === null || k <= 0) return null

  let p1 = head, p2 = head
  // 向前走k-1步，则判断条件是k-1>0， 即k > 1
  while (k > 1) {
    if (p2.next !== null) {
      p2 = p2.next
      k -= 1
    } else {
      // 走到尾节点还没到k-1的位置，说明不存在
      return null
    }
  }

  while (p2.next !== null) {
    p1 = p1.next
    p2 = p2.next
  }
  return p1

}
