# 跳台阶

# 普通跳台阶

一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法（先后次序不同算不同的结果）。

跳台阶也是一道经典的动态规划问题，关于动态规划可以先看这篇文章：动态规划

根据题目，可以得知只有一级台阶，就只有一种跳法，如果有2级台阶，那么就有两种跳法：

  1. 分两次跳，每次跳一级；
  2. 一次跳两级。

当台阶数为n并且大于2时，第一次跳有两种不同的选择：

  1. 第一次只跳一级，此时跳法数目等于后面的剩下n-1台阶跳法数目，即f(n-1)
  2. 第二种选择时第一次跳2级台阶，此时跳法数目等于剩下n-2台阶跳法数目，即f(n-2)

所有当台阶为n时，跳法总数为f(n) = f(n-1) + f(n-2)。得到该方程，我们就可以得出代码如下

function jumpFloor(number) {
  if (number <= 2) {
    return number
  }
  return jumpFloor(number - 1) + jumpFloor(number - 2)
}

这是最常规的递归做法，既然我们已经有个这个方程，那么就可以直接用动态规划的方式去实现

function jumpFloor(number)
{
    if (number === 1) return 1
    if (number === 2) return 2
    let cur = 1
    let prev = 2
    for (let i = 3; i <= number; i++) {
        // 这一步相当于f(n) = f(n-1) + f(n-2)
        let res = cur + prev
        cur = prev
        prev = res
    }
    return prev
}

# 变态跳台阶

一只青蛙一次可以跳上1级台阶，也可以跳上2级……它也可以跳上n级。求该青蛙跳上一个n级的台阶总共有多少种跳法。

这道题是跳台阶的升级版，应该也是可以通过递归或动态规划去实现的，不过观察题目，我们可以发现，可以通过计算2的幂次方得出答案，代码如下

function jumpFloorI(number) {
  if (number === 1) return 1
  return Math.pow(2, number -1)
}
