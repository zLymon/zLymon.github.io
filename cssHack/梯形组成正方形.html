  • 梯形与正方形组成一个新的正方形

这个问题是我在掘金上看到的一个别人公司的需求，如何实现一个正方形里有四个梯形围着一个正方形，并且需要点击每个块能够变色。大致的图像是这样的。

图1

首先可以先把这个正方形挪开，看一下最外层的样式是怎样的。

图2

可以发现去掉正方形后，是四个三角形形成的一个正方形，那么三角形怎么实现呢？有两种方法：

  1. 通过一个div，设置它的border属性，将其他三个边的颜色都改成transparent，只留一个border的颜色，就可以形成一个三角形。
  2. 用一个正方形，旋转45度后，遮挡住半边，也可以得到一个三角形。

因为前面说到需要实现点击每个块要变色，那么通过修改border来实现三角形就无法实现点击变色了，所以在这里用的是第二种办法。

但是在这个正方形内有四个三角形，难道要一个个地去旋转得到四个三角形吗，这样是可以实现的，但是样式会很复杂，调试起来很麻烦。我们可以通过四个正方形组成一个大的正方形，再旋转最外层的正方形后，遮挡住多余的部分，就可以得到像图二的样式了。

再整理一下实现的思路：

  1. 用四个正方形组成一个大的正方形；
  2. 对最外层的一个正方形旋转45度后，在这个正方形外面套多一个div，用于遮挡旋转后多余的部分，然后就能够得到图二的样式；
  3. 通过绝对定位在里面添加一个小的正方形，遮挡住角后则变相得到了四个梯形，最后实现了图一的样式。

源码如下

<div class="wrapper">
  <div class="container">
    <div class="div1"></div>
    <div class="div2"></div>
    <div class="div3"></div>
    <div class="div4"></div>
    <div class="div5"></div>
  </div>
</div>
<style scoped>
.wrapper {
  position: relative;
  width: 210px;
  height: 210px;
  margin: 25px auto;
  overflow: hidden;
  border: 1px solid black;
}
.container {
  position: absolute;
  top: 0;
  /* 这里的left属性需要调整成负值，因为要遮挡旋转后左右两边的角 */
  left: -45px;
  right: 0;
  bottom: 0;
  margin: auto;
  width: 300px;
  height: 300px;
  transform: rotate(45deg);
  font-size: 0;
  box-sizing: border-box;
}
div {
  display: inline-block;
  width: 150px;
  height: 150px;
}
.div1 {
  outline: black solid 1px;
}
.div4 {
  outline: black solid 1px;
}
.div5 {
  position: absolute;
  width: 100px;
  height: 100px;
  left: 150px;
  top: 150px;
  transform: translate(-50%, -50%) rotate(45deg);
  background-color: blue;
}
</style>
